## Color Coding System

### Quality Scores
```
🟢 Excellent (8-10):  Green background, dark green text
🔵 Good (6-7):        Blue background, dark blue text
🟡 Moderate (4-5):    Yellow background, dark yellow text
🔴 Low (1-3):         Red background, dark red text
```

### Difficulty Levels
```
🟢 Beginner:         Easy to print, high success rate
🔵 Intermediate:     Standard complexity, good success rate
🟡 Advanced:         Complex features, requires experience
🔴 Expert:           Very difficult, specialized knowledge needed
```

### Compatibility Status
```
✓ Compatible:        Green checkmark, supported printer
✗ Incompatible:      Red X, not supported
⚠ Partial:           Yellow warning, limited support
```

```
Desktop      Tablet       Mobile       Small
(>1024px) → (768-1024) → (<768px) → (<480px)

[4 cols]  →  [2-3 cols] → [2 cols]  → [1 col]

Detailed sections:
[2 cols]  →  [1 col]   → [1 col]   → [1 col]
```

## Usage Example Flow

```
User Action: Click 3D file
     │
     ▼
Modal Opens
     │
     ├─► Show thumbnail & basic info (immediate)
     │
     ├─► Show loading spinner
     │
     ▼
API Call: /api/v1/files/{id}/metadata/enhanced
     │
     ├─► Check cache (5min TTL)
     │   │
     │   ├─► Cache Hit → Return cached data
     │   └─► Cache Miss → Query backend
     │
     ▼
Backend Processing
     │
     ├─► Extract metadata from 3MF/G-code
     ├─► Calculate costs
     ├─► Assess quality
     └─► Check compatibility
     │
     ▼
Return JSON Response
     │
     ▼
Frontend Rendering
     │
     ├─► Render summary cards (first)
     ├─► Render detailed sections (sequential)
     └─► Apply animations & transitions
     │
     ▼
User Views Complete Metadata
     │
     ├─► Scroll through sections
     ├─► Make informed decisions
     └─► Download or adjust settings
```

---

**Visual Design Philosophy:**
- Clean and modern card-based layout
- Information hierarchy with summary → details
- Color coding for quick scanning
- Responsive and touch-friendly
- Accessible to all users
- Performance-optimized rendering
