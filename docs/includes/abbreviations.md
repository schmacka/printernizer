<!-- Format: *[ABBREVIATION]: Full definition or explanation -->

<!-- Web Technologies -->
*[API]: Application Programming Interface - A set of rules and specifications for software components to communicate
*[REST]: Representational State Transfer - An architectural style for designing networked applications
*[HTTP]: Hypertext Transfer Protocol - The foundation of data communication on the web
*[HTTPS]: Hypertext Transfer Protocol Secure - HTTP with encryption via TLS/SSL
*[WebSocket]: A communication protocol providing full-duplex communication channels over a single TCP connection
*[URL]: Uniform Resource Locator - The address of a resource on the internet
*[JSON]: JavaScript Object Notation - A lightweight data interchange format
*[XML]: eXtensible Markup Language - A markup language for encoding documents
*[CORS]: Cross-Origin Resource Sharing - A mechanism that allows restricted resources on a web page to be requested from another domain
*[TLS]: Transport Layer Security - A cryptographic protocol for secure communications
*[SSL]: Secure Sockets Layer - A cryptographic protocol (predecessor to TLS)
*[JWT]: JSON Web Token - A compact URL-safe means of representing claims between two parties

<!-- Networking -->
*[MQTT]: Message Queuing Telemetry Transport - A lightweight messaging protocol for IoT devices
*[TCP]: Transmission Control Protocol - A connection-oriented communications protocol
*[UDP]: User Datagram Protocol - A connectionless communications protocol
*[IP]: Internet Protocol - The principal communications protocol for routing data across networks
*[DNS]: Domain Name System - The system that translates domain names to IP addresses
*[FTP]: File Transfer Protocol - A standard protocol for transferring files over a network
*[SSH]: Secure Shell - A cryptographic network protocol for secure remote access
*[mDNS]: Multicast DNS - A protocol that resolves hostnames to IP addresses within small networks
*[SSDP]: Simple Service Discovery Protocol - A network protocol for discovering network services

<!-- Backend & Frameworks -->
*[FastAPI]: A modern, fast web framework for building APIs with Python based on standard Python type hints
*[ASGI]: Asynchronous Server Gateway Interface - A spiritual successor to WSGI for asynchronous Python web servers
*[WSGI]: Web Server Gateway Interface - A specification for web servers to forward requests to Python web applications
*[Uvicorn]: A lightning-fast ASGI server implementation
*[Gunicorn]: Green Unicorn - A Python WSGI HTTP server for Unix systems
*[ORM]: Object-Relational Mapping - A technique for converting data between incompatible type systems

<!-- Databases -->
*[SQLite]: A C-language library that implements a small, fast, self-contained SQL database engine
*[SQL]: Structured Query Language - A standard language for managing relational databases
*[NoSQL]: Not Only SQL - A category of database management systems that don't use relational tables
*[PostgreSQL]: An open-source relational database management system emphasizing extensibility and SQL compliance
*[MySQL]: An open-source relational database management system
*[Redis]: An in-memory data structure store used as a database, cache, and message broker
*[CRUD]: Create, Read, Update, Delete - The four basic operations of persistent storage

<!-- User Interface -->
*[UI]: User Interface - The means by which a user interacts with an application
*[GUI]: Graphical User Interface - A visual interface that uses icons, menus, and a mouse
*[CLI]: Command Line Interface - A text-based interface for interacting with software

<!-- 3D Printing -->
*[STL]: STereoLithography - A file format for 3D models used in 3D printing
*[3MF]: 3D Manufacturing Format - A modern 3D printing file format
*[GCODE]: G-Code - A language for controlling CNC machines and 3D printers
*[BGCODE]: Binary G-Code - A compressed, binary format for G-code files used by Bambu Lab printers
*[PrusaLink]: Prusa's web interface and API for remote printer control and monitoring
*[FDM]: Fused Deposition Modeling - A 3D printing process that builds objects layer by layer by extruding material
*[SLA]: Stereolithography - A 3D printing process using UV light to cure liquid resin
*[DLP]: Digital Light Processing - A 3D printing process similar to SLA but using a digital projector
*[PLA]: Polylactic Acid - A biodegradable thermoplastic used in 3D printing
*[ABS]: Acrylonitrile Butadiene Styrene - A common thermoplastic polymer used in 3D printing
*[PETG]: Polyethylene Terephthalate Glycol - A durable thermoplastic used in 3D printing
*[TPU]: Thermoplastic Polyurethane - A flexible material used in 3D printing

<!-- DevOps & Deployment -->
*[Docker]: A platform for developing, shipping, and running applications in containers
*[CI/CD]: Continuous Integration/Continuous Deployment - Automated software development practices
*[NAS]: Network Attached Storage - File-level computer data storage connected to a network
*[RPM]: Red Hat Package Manager - A package management system for Linux distributions
*[DEB]: Debian package - A software package format used by Debian and Ubuntu Linux distributions

<!-- Security & Compliance -->
*[GDPR]: General Data Protection Regulation - EU regulation on data protection and privacy
*[VAT]: Value Added Tax - A consumption tax levied on products at each stage of production
*[OAuth]: Open Authorization - An open standard for access delegation
*[AES]: Advanced Encryption Standard - A symmetric encryption algorithm
*[RSA]: Rivest-Shamir-Adleman - An asymmetric cryptographic algorithm
*[MD5]: Message Digest Algorithm 5 - A widely used cryptographic hash function (now considered insecure)
*[SHA]: Secure Hash Algorithm - A family of cryptographic hash functions

<!-- Other Technical Terms -->
*[UUID]: Universally Unique Identifier - A 128-bit number used to identify information in computer systems
*[CSV]: Comma-Separated Values - A simple file format for storing tabular data
*[YAML]: YAML Ain't Markup Language - A human-readable data serialization language
*[TTY]: Teletypewriter - A terminal device for entering data and displaying output
*[NGINX]: A high-performance web server and reverse proxy server
*[Apache]: Apache HTTP Server - An open-source web server software
*[WebRTC]: Web Real-Time Communication - A technology for peer-to-peer communication in web browsers
*[SMTP]: Simple Mail Transfer Protocol - An internet standard for email transmission
*[POP3]: Post Office Protocol version 3 - A protocol for retrieving email from a server
*[IMAP]: Internet Message Access Protocol - A protocol for accessing email on a remote server
