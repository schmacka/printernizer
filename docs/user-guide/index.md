## Overview

Printernizer provides a professional interface for managing your 3D printer fleet. Whether you're managing a single printer or multiple printers across different models (Bambu Lab A1, Prusa Core One), Printernizer gives you the tools you need to:

- Monitor printer status in real-time
- Track print jobs from start to finish
- Manage files across all your printers
- Analyze usage and performance metrics
- Automate workflows with the API
